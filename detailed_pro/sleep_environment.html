질높은 수면을 위한 주변환경 제어

Project information

  • Category: IoT, Embedded System
  • Tech Stack: Python, C/C++, MySQL, Flask
  • Tools: Arduino IDE, VS Code, NodeMCU, DHT11, LCD, 부저 등
  • Features: 수면 데이터 분석 및 환경 자동 제어
  • Database: Sleep-ACCEL 기반 수면 단계 DB 설계 및 연동
  • Duration: 약 2주일
  • Project URL: Repository

핵심 기능

  • PhysioNet 수면 데이터 전처리 및 CSV → MySQL DB 저장
  • Flask 서버를 통한 NodeMCU ↔ 아두이노 간 시리얼 통신
  • 온습도, 조도 등 센서 데이터 수집 및 LCD 디스플레이 출력
  • DB 기반 사용자 수면 상태 분석 후 맞춤형 환경 제어
  • 알람/메시지 출력 조건 설정 (D+R, FS, RT 기준)
  • 부저, LED, LCD를 활용한 상황별 수면 환경 안내
