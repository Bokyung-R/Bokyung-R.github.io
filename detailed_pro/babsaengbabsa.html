메뉴추천 애플리케이션

안드로이드로 제작한 메뉴추천 및 근처 가게 추천 앱

Project information

  • Category: Android Application
  • Tech Stack: Kotlin, Neo4j
  • Tools: Android Studio
  • Features: 사용자 취향 기반 메뉴 추천, 위치 기반 맛집 제공
  • Data Source: 사용자 설문 및 Neo4j 그래프 데이터
  • Team: 김정윤, 류보경, 이민희

핵심 기능

  • 사용자 설문을 기반으로 선호하는 메뉴 속성 파악
  • Neo4j를 활용한 그래프 관계 설계를 통한 메뉴 매칭
  • 추천된 메뉴와 연결된 음식점 위치 지도에 표시
  • 랜덤 추천 기능도 함께 제공
  • 사용자 취향 반영으로 기존 랜덤 추천의 단점 보완
