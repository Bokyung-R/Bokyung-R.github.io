미세먼지 감지 자동 창문 개폐

Project information

  • Category: IoT / Embedded System
  • Tech Stack: Arduino, 3D Printing
  • Tools: Arduino IDE, TinkerCAD, Cura
  • Features: 미세먼지 농도에 따라 자동으로 개폐되는 환기창
  • Hardware: 아두이노, 서보모터, 미세먼지 센서
  • Team: 3DUNO (6명)
  • Duration: 약 3개월

핵심 기능

  • 미세먼지 센서를 통해 실내 공기 상태 측정
  • 미세먼지 농도(PM 수치)에 따라 서보모터 제어
  • 좋음(80 이하): 창문 열림 / 보통(150 이하): 절반 개방 / 나쁨(500 초과): 창문 닫힘
  • LCD로 실시간 미세먼지 수치 표시
  • 3D 프린터를 이용해 창문 모형 및 기어 설계
